# Task 1: Sales Data Analysis 📊

This repository contains the documentation, source code, and visual insights for **Task 1 (Sales Data Analysis)** as part of the Data Analytics Internship at **Intern Alpha**.

---

## 📌 Project Objective
The primary objective of this project is to analyze retail sales data using Python to identify key revenue drivers, seasonal trends, and product performance. This analysis aids data-driven decision-making by revealing:
1. Which product lines generate the highest revenue.
2. How sales revenue fluctuates monthly (seasonal variations).
